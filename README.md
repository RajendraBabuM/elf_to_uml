# elf_to_uml
create uml file from elf file for avr

--------------------------------------------------------------------------
Install 
--------------------------------------------------------------------------
sudo ./install.sh
--------------------------------------------------------------------------


--------------------------------------------------------------------------
dependencies
--------------------------------------------------------------------------
1. avr-objdump
2. plantuml 
--------------------------------------------------------------------------

--------------------------------------------------------------------------
run command
--------------------------------------------------------------------------
 $ umlcr mainapp.elf
 
 $ umldiag recv_cb(function_name)
--------------------------------------------------------------------------





--------------------------------------------------------------------------
Note: if you donot have plantuml in your system 
you can manually open umlfile from /tmp/uml/flow_diagram/function_name.uml
--------------------------------------------------------------------------
